Functional Requirements
1. Announcements Management

Admin should be able to:

View all announcements
Add new announcements
Edit existing announcements
Delete announcements
Set publish date and expiry date
Mark announcements as active/inactive
Categorize announcements such as General, Holiday, Event, Urgent, Course-specific
2. Calendar Management

Admin should be able to:

View calendar events in monthly and list format
Add, edit, and delete calendar events
Create events for classes, holidays, exams, temple events, and special programs
Set event date, time, description, location, and category
Mark recurring events where needed

4. Teacher Assignment

Admin should be able to:

View list of teachers
Assign teachers to specific course levels/classes
Reassign teachers
See teacher name, email, phone, assigned course, assigned level, and class timing
Prevent duplicate assignment conflicts for same time slot
Show teacher-course mapping in a simple table
